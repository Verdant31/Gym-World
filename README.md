&nbsp;

## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Verdant31/Gym-World.git
```

```
# Acesse a pasta da aplicação
$ cd Gym-World

# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn start

```
